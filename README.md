# Nano Banana Image Upscaler

AI-powered image enhancement web app using Real-ESRGAN technology.

## Features

- Upload images (JPEG, PNG, WebP up to 10MB)
- AI upscaling (2x or 4x)
- Noise reduction
- Drag & drop support
- Download enhanced images

## Setup

1. Get your API key from [nanobananaguide.ai/api](https://nanobananaguide.ai/api)

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the dev server:
   ```bash
   npm run dev
   ```

4. Open http://localhost:3000 and enter your API key

## Tech Stack

- React + Vite
- Tailwind CSS
- Nano Banana API
